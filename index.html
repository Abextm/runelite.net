Runelite is a free open source (BSD-2 licensed) RuneScape 2 client and deobfuscation project aimed at developers wanting to learn. Its source tree includes a deobfuscated client as well as a developer client which includes plugins for additional funtionality.

Runelite uses Maven as its build system, and so will compile and run out of the box in any environment which supports it. Netbeans works well, and we have a wiki article for building and running Runelite with IntelliJ IDEA.

You may download the launcher above, which will download the latest release to ~/.runelite and launch it.

If you have any questions please join our Discord server.
